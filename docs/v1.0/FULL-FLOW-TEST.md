## 8. 失败分级与处理


| 级别     | 现象                     | 建议动作               |
| ------ | ---------------------- | ------------------ |
| **P0** | 无法启动、无法保存项目、生成静默无落盘    | 阻断发布；附 `main` 进程日志 |
| **P1** | 四路生成任一完全不可用；助手无法连接     | 阻断发布               |
| **P2** | 熔断 UX 异常、续生成覆盖已有章、导出失败 | 修复或记入 Known Issues |
| **P3** | 文案/样式、非关键提示不准确         | 可延后                |


**日志位置**：

- 开发：`npm run dev` 终端 + DevTools Console
- Generation Console：工作区底部面板
- Windows 用户数据：`%APPDATA%/novels-creator/`（配置与 secrets）

**curl 复现 Dify Legacy**（仅 Legacy 路径）：

```bash
curl -X POST "http://127.0.0.1/v1/workflows/run" \
  -H "Authorization: Bearer app-你的Key" \
  -H "Content-Type: application/json" \
  -d @dify/chapter/fixtures/run-request-success.json
```
